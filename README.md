## DeepSpell


Installation
Prerequisites:
```sh
Python 3.8, virtualenv, pip installed
python3 -m virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

To read more about the Seq2seq models, follow the [link](https://github.com/bentrevett/pytorch-seq2seq/blob/master/1%20-%20Sequence%20to%20Sequence%20Learning%20with%20Neural%20Networks.ipynb).
